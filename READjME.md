## Featured Project

### ЁЯПЖ Pitch Control тАФ Probabilistic Football Analytics
*FIFA / DFL / Sportradar Competition ┬╖ January 2025*

Led a 6-person team building a physics-ML hybrid system using professional DFL tracking data and AWS infrastructure. The core challenge: compute real-time pass success probability across all 22 players simultaneously.

- **Physics layer** тАФ Langevin Dynamics for player movement + Velocity Verlet integration for 3D ball trajectory with aerodynamic drag modeling (`< 2.0m` pass destination precision)
- **ML layer** тАФ XGBoost classifier fed with physics-derived features (Time-to-Intercept, Control Volume) and limb tracking data, achieving `> 0.90 ROC AUC`
- **Infrastructure** тАФ AWS SageMaker + S3 pipeline processing 25 Hz optical tracking data at scale
- **Novel contribution** тАФ First worldwide integration of limb tracking data into pass prediction models
